JAVA GUI 게임

- SWING, AWT
- 부족전쟁 같은 게임 
- 기간 : 2020.01 ~ 2020.02

## 게임 설명

- 에너지를 소모하여 생성한 소환수가 적 캐릭터를 공격
- 플레이어 혹은 적의 체력이 0이 되면 승리 / 패배
- 상점에서 스킬, 포션 구입 가능

- 메이플스토리로부터 허용받은 범위에서 이미지, 음악 소스 사용
